# Sandbox And Escalation Decision Tree

最后更新时间：YYYY-MM-DD

本清单用于减少“先撞沙箱，再补边界说明”的返工。

## 先判断任务类型

- 当前路径是否在 `<WORKSPACE_ROOT>` 内。
- 当前动作是只读检查，还是会写文件、跑网络、访问全局配置或外部服务。
- 当前命令是否涉及中文路径、递归扫描、并行读取或工作区外目录。

## 遇到 sandbox/helper 错误时

先不要把它当作文件损坏。按顺序排查：

1. 改成绝对路径。
2. 改成分目录、小范围读取，不要直接递归或高并发。
3. 判断当前动作是否只读。
4. 判断当前路径是否在工作区外、是否属于全局配置、是否涉及网络/认证、Git 写入或破坏性操作。
5. 如果目标在 `<WORKSPACE_ROOT>` 内且动作可由普通沙箱完成，不要因为一次 helper 抖动就申请提权；优先换成单条命令、受控写回或更小范围操作。
6. 只有工作区内降级仍无法完成，或动作本身需要网络/认证/Git 写入/工作区外访问时，再按权限流程申请提权。

## 什么时候应预判提权

- 读取或修改工作区外的全局配置、全局 skill、系统 skill、全局 `AGENTS.md`。
- 访问网络、远端仓库、外部 API、云资源或真实发送通道。
- 命令需要处理工作区外进程、缓存、认证或环境配置。
- Git 写入索引、创建提交、更新 refs、push/pull/fetch 等会触发仓库状态或网络变化的操作。

## 提权前要说清楚

- 访问或修改的具体路径
- 为什么必须访问该路径
- 这是只读检查还是会写入
- 对当前工作区的影响范围


## 降级与提权判定表

| 信号 | 先做 | 再做 | 不要做 |
|---|---|---|---|
| 只读命令出现 helper 抖动 | 绝对路径 + 单文件读取 | 工作区内继续未提权降级；工作区外才申请只读提权 | 反复并行重试 |
| `apply_patch` 在长中文 Markdown 中断 | 检查文件是否部分落盘 | 工作区内使用 Windows safe editing 受控写回 | 未确认状态就继续补丁或直接提权 |
| 中文路径读取失败 | 缩小路径范围 | 只读提权说明 sandbox 降级 | 判断文件损坏 |
| Git/网络/外部 API 失败 | 区分 sandbox、认证、网络和远端错误 | 需要网络或认证时申请提权 | 把 API 成功等同用户可见 |
| 工作区外全局配置 | 说明路径、目的和影响 | 获得用户批准后再读写 | 默认直接修改 |

提权不是默认第一步；只有工作区外路径、网络/认证、真实外部发送，或普通沙箱已出现明确 helper 抖动且必要动作无法完成时，才进入提权流程。

## 常见错误归因

- `helper_unknown_error` / `setup refresh had errors`：优先判断路径、并发和沙箱限制，不要直接判断项目文件损坏。
- 中文路径下 `apply_patch` 或读取失败：先缩小范围，再考虑改用受控写回或提权只读。
- 网络请求失败：先区分代理、沙箱网络权限、认证和真实服务异常。
## Windows sandbox/helper 抖动降级策略

适用信号：`windows sandbox: helper_unknown_error`、`setup refresh had errors`、`orchestrator_helper_launch_canceled`、中文路径下普通读取失败、长 Markdown 补丁卡住或被中断。

默认处理顺序：

1. 停止连续硬重试，不把错误归因为文件损坏。
2. 先确认目标路径和文件是否存在；优先使用绝对路径。
3. 将递归扫描、并行读取、大范围 `Get-Content` 改成小范围单文件读取。
4. 只读检查如果仍失败，先确认是否在工作区内；工作区内优先继续用更小范围的未提权读取，工作区外才按权限流程使用 `require_escalated` 只读读取。
5. 长中文 Markdown 或治理文件写入时，如果 `apply_patch` 首次失败、卡住或被中断，先读取文件头/关键段确认当前状态，再改用 Windows safe editing 的 UTF-8 no BOM 受控写回；工作区内不得因首次 helper 抖动直接提权。
6. 写回后必须做文件头、文件尾、关键词、旧口径残留和异常控制字符复核。

不要做：

- 不要在 helper 抖动后反复并行重试同一批命令。
- 不要在未确认文件状态时宣称“已完成”。
- 不要把提权写入当作默认第一步；除非路径在工作区外、需要网络/认证、或普通沙箱已经出现明确 helper 抖动。
